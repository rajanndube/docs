## What is Stringboot?

Stringboot is a powerful internationalization (i18n) SDK that provides high-performance string localization for modern applications across Web, Android, and iOS platforms. It features smart multi-layered caching, offline support, and reactive string updates with minimal overhead.

### Key Features

<CardGroup cols={2}>
  <Card title="Offline-First" icon="wifi-slash">
    Works seamlessly without network connectivity using smart caching
  </Card>
  <Card title="Smart Caching" icon="database">
    Multi-layer caching with LRU memory management and persistent storage
  </Card>
  <Card title="Delta Sync" icon="arrows-rotate">
    Only downloads changed strings using String-Sync v2 protocol
  </Card>
  <Card title="Fast Lookups" icon="bolt">
    Target &lt;300ms string lookups on 4G networks
  </Card>
</CardGroup>

## Why Stringboot?

<AccordionGroup>
  <Accordion title="Offline-First Architecture" icon="wifi-slash">
    Stringboot is designed to work seamlessly offline. After the initial sync, all strings are cached locally, ensuring your app works perfectly without network connectivity.
  </Accordion>

  <Accordion title="Blazing Fast Performance" icon="bolt">
    Multi-layer caching ensures sub-millisecond string lookups from memory, with automatic fallback to local database and network when needed.
  </Accordion>

  <Accordion title="Bandwidth Efficient" icon="chart-line">
    Delta sync using String-Sync v2 protocol means only changed strings are downloaded. ETag-based caching prevents unnecessary network requests.
  </Accordion>

  <Accordion title="Developer Friendly" icon="code">
    Native SDKs for each platform with idiomatic APIs. React hooks for web, Flow API for Android, SwiftUI integration for iOS.
  </Accordion>
</AccordionGroup>

## Quick Example

See how easy it is to get started with Stringboot:

<CodeGroup>

```javascript Web SDK
import StringBoot from '@stringboot/web-sdk';

// Initialize
await StringBoot.initialize({
  apiToken: 'your-api-token',
  baseUrl: 'https://api.stringboot.com',
  defaultLanguage: 'en'
});

// Get a string
const welcome = await StringBoot.get('welcome_message');
console.log(welcome); // "Welcome to our app!"

// Change language
await StringBoot.changeLanguage('es');
// Strings automatically update
```

```kotlin Android SDK
// Initialize in Application class
StringbootExtensions.autoInitialize(this)

// Use in XML
<TextView
    android:tag="welcome_message"
    android:text="@string/welcome_message" />

// Apply strings
binding.root.applyStringbootTags()

// Or get programmatically
val text = StringProvider.get("welcome_message")
```

```swift iOS SDK
// Initialize in App
StringProvider.shared.initialize(
    cacheSize: 1000,
    apiToken: "your-api-token",
    baseURL: "https://api.stringboot.com",
    autoSync: true
)

// Use in SwiftUI
SBText("welcome_message")
    .font(.title)
```

</CodeGroup>
