## Overview

The Stringboot Android SDK provides high-performance string localization for Android applications with offline-first architecture, smart caching, and reactive UI updates. Built with Kotlin and optimized for modern Android development.

### Core Features

<CardGroup cols={3}>
  <Card title="Dynamic Strings" icon="font">
    Update app text instantly without releasing new versions
  </Card>
  <Card title="Multi-Language" icon="language">
    Seamless translations and language switching
  </Card>
  <Card title="A/B Testing" icon="flask">
    Optimize messaging with built-in experiment tracking
  </Card>
</CardGroup>

### Platform Requirements

| Requirement     | Version          |
| --------------- | ---------------- |
| **Min SDK**     | 24 (Android 7.0) |
| **Target SDK**  | 34 (Android 14)  |
| **Kotlin**      | 1.9.22+          |
