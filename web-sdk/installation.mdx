## 1. Install Package

Install the SDK using your preferred package manager:

```bash
npm install @stringboot/web-sdk
# or
yarn add @stringboot/web-sdk
# or
pnpm add @stringboot/web-sdk
```

## 2. Initialize SDK

Initialize the SDK at the entry point of your application.

<Tabs>
  <Tab title="React / Next.js">
    ```jsx
    import { StringbootProvider } from '@stringboot/web-sdk/react';

    function App({ Component, pageProps }) {
      // Analytics Handler for A/B Testing
      const handleExperiments = (experiments) => {
        experiments.forEach(exp => logToAnalytics(exp));
      };

      return (
        <StringbootProvider
          apiToken="YOUR_API_TOKEN"
          initialLocale="en"
          onExperimentsAssigned={handleExperiments}
          deviceId="USER_ID" // Optional
          faqOptions={{ enable: true, cacheSize: 100 }}
          debug={process.env.NODE_ENV === 'development'}
        >
          <Component {...pageProps} />
        </StringbootProvider>
      );
    }
    ```
  </Tab>
  <Tab title="Vue.js">
    ```javascript
    app.use(StringbootVue, {
      apiToken: 'YOUR_API_TOKEN',
      initialLocale: 'en',
      analyticsHandler: (exps) => { ... },
      deviceId: 'USER_ID',
      faq: { enabled: true },
      debug: true
    });
    ```
  </Tab>
  <Tab title="Vanilla JS">
    ```javascript
    import { Stringboot, FAQProvider } from '@stringboot/web-sdk';

    await Stringboot.init({
      apiToken: 'YOUR_API_TOKEN',
      initialLocale: 'en',
      analyticsHandler: (exps) => { ... },
      deviceId: 'USER_ID',
      debug: true
    });

    // Initialize FAQs separately
    await FAQProvider.init({ apiToken: 'YOUR_API_TOKEN' });
    ```
  </Tab>
</Tabs>

<Note>
  Replace `YOUR_API_TOKEN` with your actual API token from the Stringboot Dashboard.
</Note>
