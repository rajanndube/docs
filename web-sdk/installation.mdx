## Installation

Choose your preferred package manager to install the Stringboot Web SDK.

<CodeGroup>

```bash npm
npm install @stringboot/web-sdk
```

```bash yarn
yarn add @stringboot/web-sdk
```

```bash pnpm
pnpm add @stringboot/web-sdk
```

</CodeGroup>

## Prerequisites

- **Modern Browser:** ES2020+ support required (Chrome 87+, Firefox 78+, Safari 14+, Edge 88+)
- **API Token:** [Obtain from your Stringboot dashboard](https://dashboard.stringboot.com)
- **Node.js:** Version 18+ (for development/build tools only)

## Quick Start

### Vanilla JavaScript

Initialize the SDK in your application entry point:

```javascript
import StringBoot from '@stringboot/web-sdk';

// Initialize the SDK
await StringBoot.initialize({
  apiToken: import.meta.env.VITE_STRINGBOOT_TOKEN,
  baseUrl: 'https://api.stringboot.com',
  defaultLanguage: 'en',
  debug: false
});
```

### React

Use the `useStringBoot` hook in your root component:

```jsx
import { useStringBoot } from '@stringboot/web-sdk/react';

export function App() {
  const { initialized, error } = useStringBoot({
    apiToken: import.meta.env.VITE_STRINGBOOT_TOKEN,
    baseUrl: 'https://api.stringboot.com',
    defaultLanguage: 'en',
    debug: false
  });

  if (!initialized) return <LoadingScreen />;
  if (error) return <ErrorScreen error={error} />;

  return <MainApp />;
}
```

<Tip>
Initialize the SDK as early as possible in your application to ensure strings are available when needed.
</Tip>

## Configuration Options

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `apiToken` | string | Yes | Your API token from the Stringboot dashboard |
| `baseUrl` | string | No | API endpoint (default: `https://api.stringboot.com`) |
| `defaultLanguage` | string | No | Default language code (default: `en`) |
| `debug` | boolean | No | Enable debug logging (default: `false`) |
| `enableIntegrityCheck` | boolean | No | Enable integrity verification for strings (default: `false`) |

## Environment Variables

Store your API token securely using environment variables:

<CodeGroup>

```javascript Vite
// .env
VITE_STRINGBOOT_TOKEN=your_api_token_here

// main.js
const config = {
  apiToken: import.meta.env.VITE_STRINGBOOT_TOKEN,
  baseUrl: 'https://api.stringboot.com',
  defaultLanguage: 'en',
  debug: import.meta.env.DEV
};
```

```javascript Create React App
// .env
REACT_APP_STRINGBOOT_TOKEN=your_api_token_here

// index.js
const config = {
  apiToken: process.env.REACT_APP_STRINGBOOT_TOKEN,
  baseUrl: 'https://api.stringboot.com',
  defaultLanguage: 'en',
  debug: process.env.NODE_ENV === 'development'
};
```

```javascript Next.js
// .env.local
NEXT_PUBLIC_STRINGBOOT_TOKEN=your_api_token_here

// _app.js
const config = {
  apiToken: process.env.NEXT_PUBLIC_STRINGBOOT_TOKEN,
  baseUrl: 'https://api.stringboot.com',
  defaultLanguage: 'en',
  debug: process.env.NODE_ENV === 'development'
};
```

</CodeGroup>

<Warning>
Never commit API tokens to source control. Always use environment variables and add `.env` files to your `.gitignore`.
</Warning>

## Complete React Example

Here's a production-ready React setup with proper loading and error states:

```jsx
import React from 'react';
import { useStringBoot, useString } from '@stringboot/web-sdk/react';

export function App() {
  // Initialize SDK with environment variables
  const { initialized, error } = useStringBoot({
    apiToken: import.meta.env.VITE_STRINGBOOT_TOKEN,
    baseUrl: 'https://api.stringboot.com',
    defaultLanguage: 'en',
    debug: import.meta.env.DEV
  });

  // Show loading state during initialization
  if (!initialized) {
    return (
      <div className="loading">
        <div className="spinner" />
        <p>Loading...</p>
      </div>
    );
  }

  // Show error state if initialization fails
  if (error) {
    return (
      <div className="error">
        <h1>Initialization Error</h1>
        <p>{error}</p>
      </div>
    );
  }

  // Render main application once initialized
  return <MainApp />;
}

function MainApp() {
  const title = useString('app_title');
  const welcomeMsg = useString('welcome_message');

  return (
    <div>
      <h1>{title}</h1>
      <p>{welcomeMsg}</p>
    </div>
  );
}
```

## Complete Vanilla JavaScript Example

Here's a production-ready vanilla JavaScript setup:

```javascript
import StringBoot from '@stringboot/web-sdk';

async function initApp() {
  try {
    // Initialize SDK
    await StringBoot.initialize({
      apiToken: import.meta.env.VITE_STRINGBOOT_TOKEN,
      baseUrl: 'https://api.stringboot.com',
      defaultLanguage: 'en',
      debug: import.meta.env.DEV
    });

    // Set up reactive watchers
    StringBoot.watch('app_title', (value) => {
      document.getElementById('title').textContent = value;
    });

    StringBoot.watch('welcome_message', (value) => {
      document.getElementById('welcome').textContent = value;
    });

    console.log('✓ SDK initialized successfully');

  } catch (error) {
    console.error('✗ Initialization failed:', error);
    showErrorState(error.message);
  }
}

// Initialize on page load
initApp();
```
