Stringboot's A/B testing engine works seamlessly on the web.

## How It Works



<Steps>
  <Step title="Assignment">
    The SDK assigns a unique Device ID (stored in localStorage) and determines the user's variant.
  </Step>
  <Step title="Delivery">
    `useString` and `Stringboot.get()` automatically return the variant string.
  </Step>
  <Step title="Tracking">
    You log the assignment to your analytics provider.
  </Step>
</Steps>

## Analytics Integration

To track experiment performance, you need to listen to the `experiment_assigned` event.

<Steps>
  <Step title="Implement Analytics Handler">
    Pass a callback function to handle experiment assignments. Choose your analytics provider below:

    <Tabs>
      <Tab title="Google Analytics 4">
        ```javascript
        const analyticsHandler = (assignment) => {
          if (window.gtag) {
            gtag('event', 'experiment_assigned', {
              experiment_key: assignment.experimentKey,
              variant_name: assignment.variantName,
              string_key: assignment.stringKey
            });
          }
        };
        ```
      </Tab>
      <Tab title="Mixpanel">
        ```javascript
        const analyticsHandler = (assignment) => {
          if (window.mixpanel) {
            mixpanel.track('Experiment Assigned', {
              experiment_key: assignment.experimentKey,
              variant_name: assignment.variantName,
              string_key: assignment.stringKey
            });
          }
        };
        ```
      </Tab>
      <Tab title="Segment">
        ```javascript
        const analyticsHandler = (assignment) => {
          if (window.analytics) {
            analytics.track('Experiment Assigned', {
              experimentKey: assignment.experimentKey,
              variantName: assignment.variantName,
              stringKey: assignment.stringKey
            });
          }
        };
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>

### Vanilla JS

```javascript
Stringboot.on('experiment_assigned', (experiment) => {
  console.log('New experiment assignment:', experiment);
});
```

## Debugging

You can inspect active experiments in the browser console:

```javascript
await Stringboot.getExperiments();
```
