Stringboot provides framework-specific integrations for easy usage.

## React

### useString Hook

The `useString` hook is the simplest way to get a string. It automatically updates when the language changes.

```jsx
import { useString } from '@stringboot/web-sdk/react';

function WelcomeMessage() {
  const { value, loading } = useString('welcome_message');

  if (loading) return <span>Loading...</span>;
  return <h1>{value}</h1>;
}
```

### String Component

You can also use the `<String />` component:

```jsx
import { String } from '@stringboot/web-sdk/react';

function Header() {
  return (
    <h1>
      <String keyName="welcome_message" />
    </h1>
  );
}
```

## Vue

### Composition API

```javascript
<script setup>
import { useString } from '@stringboot/web-sdk/vue';

const { value: welcomeText } = useString('welcome_message');
</script>

<template>
  <h1>{{ welcomeText }}</h1>
</template>
```

### Directive

```html
<h1 v-string="'welcome_message'"></h1>
```

## Vanilla JS

```javascript
import { Stringboot } from '@stringboot/web-sdk';

// Get string (async)
const text = await Stringboot.get('welcome_message');

// Subscribe to changes
Stringboot.subscribe('welcome_message', (newValue) => {
  document.getElementById('welcome').innerText = newValue;
});
```

## Changing Language

To switch the language of the app:

```javascript
import { Stringboot } from '@stringboot/web-sdk';

// Switch to Spanish
await Stringboot.setLocale('es');
```

<Note>
  In React and Vue, all components using `useString` or directives will re-render automatically with the new language.
</Note>
