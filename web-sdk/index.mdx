## Overview

The Stringboot Web SDK provides high-performance string localization for web applications with offline-first architecture, IndexedDB persistence, and optimized React hooks for seamless integration.

### Core Features

<CardGroup cols={3}>
  <Card title="Dynamic Strings" icon="font">
    Update app text instantly without releasing new versions
  </Card>
  <Card title="Multi-Language" icon="language">
    Seamless translations and language switching
  </Card>
  <Card title="A/B Testing" icon="flask">
    Optimize messaging with built-in experiment tracking
  </Card>
</CardGroup>
